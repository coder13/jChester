jfly/jChester

jChester is a jQuery plugin designed specifically for entering speedsolving times. It currently uses Bootstrap 3 for styling, but the plan is to make it standalone in the near future.

jChester deals with SolveTimes, which have the following format: 

{
  millis: Integer,           // The number of milliseconds (after any penalties have been applied).
  decimals: Integer,         // The number of decimal places of accuracy in millis, when treated a number of seconds in base 10.
  penalties: [ String ],     // An array of penalties applied to the solve. No penalties are currently defined, but may be in the future.
  puzzlesSolved: Integer,    // The number of puzzles solved in this SolveTime. If puzzlesSolved is 0, this SolveTime is a DNF (see also multiple blindfolded rules). If not set, puzzlesSolved should be treated as 1.
  puzzlesAttempted: Integer, // The number of puzzles attempted in this SolveTime. Must be >= puzzlesSolved. If puzzlesAttempted is 0, this SolveTime is a DNS. If not set, puzzlesAttempted should be treated as 1.
  moveCount: Integer,        // An integer number of moves. Used for FMC.
}

<script>
  $(function() {
    var $inputTime = $("div.jChester").jChester({
      solveTime: $.stopwatchFormatToSolveTime("1:4.242")
    });
    $inputTime.on("solveTimeInput", function(e, solveTime) {
      console.log(solveTime);
    });
    $inputTime.on("solveTimeChange", function(e, solveTime) {
      console.log(solveTime);
    });
  });
</script>

Methods

Method Description
$('div').jChester(options) Initialize jChester on a jQuery compenent (must be a div!) if jChester is not already initialized on it.
Option Description
solveTime SolveTime to set the jChester to. If null, clears the jChester.
editableSolveTimeFields An object whose keys are fields from SolveTime to make editable via this jChester. Valid fields are currently millis, puzzlesSolved, puzzlesAttempted, and moveCount.
editableSolveTimeFields defaults to { millis: true }.
$('div').jChester('getSolveTime') Returns the current SolveTime, which may be null.
$('div').jChester('setSolveTime', SolveTime) Set the current SolveTime, may be null.
$.stopwatchFormatToSolveTime(String) Converts a stopwatch formatted string to a SolveTime. Throws an exception if the stopwatch format is invalid.
Example:
> $.stopwatchFormatToSolveTime("1:4.242")
{ millis: 64242, decimals: 3 }
$.solveTimeToStopwatchFormat(SolveTime) Converts a SolveTime to a pretty stopwatch formatted string.
Example:
> $.solveTimeToStopwatchFormat({ millis: 64242, decimals: 3 })
"1:4.242"

Events

Event Description
solveTimeInput Fires whenever the current SolveTime changes. The current SolveTime is passed as an extra parameter.
solveTimeChange Fires when the user confirms the current SolveTime, by pressing enter. The current SolveTime is passed as an extra parameter.
