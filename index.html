jChester

jChester is a jQuery plugin designed specifically for entering speedsolving times. It currently uses Bootstrap 3 for styling, but the plan is to make it standalone in the near future.

jChester deals with SolveTimes, which have the following format: 

{
  millis: Integer,      // The number of milliseconds (after any penalties have been applied).
  decimals: Integer,    // The number of decimal places of accuracy in millis, when treated a number of seconds in base 10.
  penalties: [ String ] // An array of penalties applied to the solve. Currently supports "DNF" and "DNS", which are mutually exclusive.
}

Demo

Source
<script>
  $(function() {
    var $inputTime = $("div.jChester").jChester({
      solveTime: {
        millis: 4242,
        decimals: 3,
      }
    });
    $inputTime.on("solveTimeInput", function(e, solveTime) {
      console.log(solveTime);
    });
    $inputTime.on("solveTimeChange", function(e, solveTime) {
      console.log(solveTime);
    });
  });
</script>

<div class="jChester">

Methods

Method Description
$('div').jChester(options) Initialize jChester on a jQuery compenent (must be a div!) if jChester is not already initialized on it. The jChester's time will be set to options.solveTime, if given.
$('div').jChester('getSolveTime') Returns the current SolveTime, which may be null.

Events

Event Description
solveTimeInput Fires whenever the current SolveTime changes. The current SolveTime is passed as an extra parameter.
solveTimeChange Fires when the user confirms the current SolveTime, by pressing enter. The current SolveTime is passed as an extra parameter.
